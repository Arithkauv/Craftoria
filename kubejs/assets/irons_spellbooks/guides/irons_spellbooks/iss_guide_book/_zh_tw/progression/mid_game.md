# 遊戲中期

## 卷軸鍛造

卷軸鍛造坊達到地獄資源等級後，您將解鎖獲得力量的全新動力：卷軸鍛造坊

卷軸鍛造坊通過使用墨水、紙張和焦點來製作卷軸。墨水決定了卷軸的稀有度（因此也決定了卷軸的等級），而焦點則決定了卷軸的流派。
