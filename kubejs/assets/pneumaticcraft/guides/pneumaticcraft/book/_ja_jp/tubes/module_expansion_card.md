# モジュール拡張カード

*モジュール拡張カード*は[チューブモジュール](./tube_modules.md)に取り付けてさまざまな方法で機能を拡張できるプラグ可能な回路基板です。*チューブモジュール上のカードを右クリック*して適用します。

モジュールの拡張方法の詳細については個々のチューブモジュールのページを参照してください。
