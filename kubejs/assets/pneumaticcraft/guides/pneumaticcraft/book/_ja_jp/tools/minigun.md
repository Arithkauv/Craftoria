# ミニガン

*ミニガン*は[圧縮空気](../base_concepts/pressure.md)と[弾薬](./minigun_ammo.md)で動作する手持ち武器です。弾薬は[ミニガンのマガジン](./minigun_ammo.md#magazine)に挿入する必要があります。プレイヤーのインベントリから消費されることはありません。

ミニガンを発射するには単に*右クリック*を押し続けます。銃身が回転し始めて速度が上がるとすぐに発射されます。

## アップグレード

*ミニガン*の多くの機能(スピンアップ速度、範囲、ポーション発動確率など)は[充填ステーション](../machines/charging_station.md)でアップグレードできます。

各アップグレードとその効果に関する完全なドキュメントは充填ステーションのアップグレードGUIで参照できます。
