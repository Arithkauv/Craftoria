# 無限ランチャー

無限ランチャーはMob監禁ツールから一定距離のエンティティを捕獲または解放できるツールです。GUIでモードを選択してランチャーを起動すると、インベントリ内のMob監禁ツールを使用してMobを解放または捕獲できるようになります。
