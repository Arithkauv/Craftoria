# Скорость Натяжения

<Color id="blue">Скорость Натяжения</Color> это то, насколько быстро вы можете использовать оружие дальнего боя по сравнению с первоначальной скоростью. 1 = 100%, 2 = 200%, и т.д.

Находится на оружии дальнего боя.
