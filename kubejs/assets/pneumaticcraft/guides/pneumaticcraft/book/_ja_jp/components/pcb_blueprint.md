# PCBブループリント

*PCBブループリント*は[UVライトボックス](../uv_light_box.md)の作成に必要な特別なコンポーネントです。

 通常の作成はできず、通常は[アマドローンタブレット](../amadron_tablet.md)を介すか[メカニックの村人](../villagers.md)を介して取引する必要があります。
