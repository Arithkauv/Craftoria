# 鉱石レーザーベース

[レーザードリル](./laser_drill.md)を完全に充電すると鉱石が生成されます。生成できる各鉱石にはマシンが配置されているバイオームと、マシンが採掘するように設定されている深さに基づいた重量があります。

レーザーレンズは生成された鉱石の重量を増加させます。

値についてはJEIを確認してください。
