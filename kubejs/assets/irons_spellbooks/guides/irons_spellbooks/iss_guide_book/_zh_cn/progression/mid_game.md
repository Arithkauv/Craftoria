# 游戏中期

## 卷轴撰写台

到达这一阶段后，你将可以解锁获得法术的全新方式：法术撰写台

制造法术需要用到墨水、纸与触媒（触媒包含绿宝石、烈焰棒、末影珍珠、雷电瓶、鲜血精华、神祈宝珠、冰封骸骨、回响碎片与毒马铃薯）墨水的品质决定法术的等级，触媒的种类决定法术的流派。
