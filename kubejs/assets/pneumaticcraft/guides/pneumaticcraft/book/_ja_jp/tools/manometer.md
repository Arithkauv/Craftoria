# 圧力計

圧力計を使用すると空気圧ネットワーク上の[圧力](../base_concepts/pressure.md)を測定できます。さらに一部の空気圧機械/チューブには特別な情報があります。たとえば、[セーフティチューブモジュール](../tubes/safety_module.md)はその安全閾値を表示します。したがって、レッドストーンの数学に慣れていない場合はこのツールが役立つ場合があります。

## その他MOD
