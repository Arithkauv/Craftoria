# 条件: アイテムウィジェット

これは[条件](./conditions.md)ウィジェットです。

*条件: アイテム*ウィジェットはインベントリにある一定量のアイテムをチェックします。一致するすべてのアイテムをカウントします(接続された[アイテムフィルター](./item_filter.md)によって指定されます。すべてのアイテムをカウントするにはフィルターを省略します)。
