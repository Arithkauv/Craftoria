# 座標演算ウィジェット

*座標演算*ウィジェットは[変数](./variables.md)を定義および/または操作して高度なドローン機能を実現するために使用されます。このウィジェットは使用する値を定義するために1つ以上の接続された[座標](./coordinate.md)ウィジェットを必要とし、それらの接続された部分に対して特定の算術演算(ウィジェットのGUIで定義)を実行します。

使用可能な演算は次のとおりです:
- $(太字)加算/減算$(/太字): 右側のウィジェットを加算し、左側のウィジェットを減算します。
- $(太字)乗算/除算$(/太字): 右側のウィジェットを乗算し、その結果を左側のウィジェットで除算します。
- $(太字)最小/最大$(/太字): 右側のウィジェットから最大のX/Y/Zを取得し、次に左側のウィジェットの最小値と以前に決定された最大値を取得します。

最終結果は座標演算GUIで入力された名前で定義された[変数](./variables.md)に保存されます。この*変数*を使用して後で座標を取得できます。
- [エリア](./area.md)ウィジェットではGPS座標の代わりに、*変数*名を使用できます。
- [条件: 座標](./condition_coordinate.md)を使用して、*変数*でテストを実行できます。
