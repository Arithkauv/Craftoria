# エンティティエクスポートウィジェット

[ドローン](../drone.md)は接続された[テキスト](./text.md)ウィジェットで指定された(オプションの)[エンティティフィルター](../entity_filter.md)によって許可されている場合、指定された[エリア](./area.md)に移動して運んでいるエンティティをドロップします。

[エンティティインポート](./entity_import.md)ウィジェットも参照してください。
