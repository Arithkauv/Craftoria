# レッドストーンモジュール

このモジュールは接続されたチューブ上の他のレッドストーンモジュールと<Color hex="#f00">レッドストーン信号</Color>を交換し、チューブをレッドストーンの導管として効果的に使用します。

モジュールのGUIまたは任意の*染料*でモジュールを右クリックすることで最大16個の独立したチャンネルを選択できます。

モジュールを*レンチ*で右クリックすると入力モードと出力モードが切り替わります。

モジュールに[モジュール拡張カード](./module_expansion_card.md)を適用すると追加の信号処理が可能になります。モジュールが*出力モード*の場合、モジュールを右クリックしてGUIを開き、出力信号に適用するモード(信号の反転、別のチャンネルとのAND/OR/XORなど)を選択できます。

各モードの効果については、モジュールのGUIで詳しく説明されています。
