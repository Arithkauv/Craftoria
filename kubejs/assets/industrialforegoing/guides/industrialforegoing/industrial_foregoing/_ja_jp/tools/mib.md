# Mob監禁ツール

監禁ツールを手に持った状態でエンティティを<Color id="gold">右クリック</Color>するとそのエンティティは監禁ツール内に格納されます。

地面にエンティティを置いた状態で<Color id="gold">右クリック</Color>するとエンティティを解放できます。
