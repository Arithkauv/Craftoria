# アップグレード

*アップグレード*は機械や一部のツールに挿入して何らかの方法で機能を強化できる作成可能なアイテムです。ツールにアップグレードを追加するには[充填ステーション](../charging_station.md)が必要です。

各アップグレードの正確な効果はデバイスによって異なります。マシンのGUIまたは(アイテムの場合)充填ステーションのアップグレードのGUIに記載されています。
