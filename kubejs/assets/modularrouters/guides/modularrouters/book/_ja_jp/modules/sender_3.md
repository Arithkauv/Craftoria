# 送信モジュールMk3

このトップエンドの送信モジュールは[送信モジュールMk2](./sender_2.md)と非常によく似ていますが、制限なしで任意のインベントリ(任意のディメンション)に送信できます！

注: ターゲットのインベントリはチャンクが読み込まれている必要があります。<Color hex="#228">Modular Routers</Color>はこれを実行しません。
