# Dano pelo PV atual

<Color id="blue">Dano pelo PV atual</Color> é um tipo de dano ao contato. A cada acerto, você causa uma porcentagem da vida atual do alvo como dano adicional.

Este atributo só é encontrado em armas pesadas.
