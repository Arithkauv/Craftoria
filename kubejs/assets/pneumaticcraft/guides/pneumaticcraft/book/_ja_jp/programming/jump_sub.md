# ジャンプサブルーチンウィジェット

このウィジェットは *ジャンプ*のように動作しますが、サブルーチンが完了した場合*ジャンプサブルーチン*ウィジェットの下部に接続されたウィジェットで実行が継続されます。*ジャンプサブルーチン*ウィジェットは共通コードを再利用し、サブルーチンにジャンプする前に変数を割り当てることで任意の引数でサブルーチンを呼び出すのに役立ちます。
