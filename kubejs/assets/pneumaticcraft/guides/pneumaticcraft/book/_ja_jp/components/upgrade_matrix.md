# アップグレード基盤

*アップグレード基盤*は[アップグレード](../base_concepts/upgrades.md)の作成時に*ラピスラズリ*の代わりに使用できます。アップグレード基盤を使用する利点はラピスラズリの使用量が大幅に減少することです。これはラピスラズリが限られていて、作成するアップグレードが多数ある場合に恩恵となる可能性があります。
